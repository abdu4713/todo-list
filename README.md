To-Do List App
A sleek, intuitive, and highly responsive To-Do List application designed to help users organize their daily tasks and boost productivity. Built entirely with clean, modern web technologies.

Features
Task Management: Easily add, edit, and delete tasks in real-time.

Status Tracking: Mark tasks as completed with a single click or tap.

Persistent Storage: Saves your list automatically using local storage so you never lose your progress.

Premium UI: Features a minimalist, dark/light mode friendly design with smooth transitions.

Fully Responsive: Optimised for seamless use across desktops, tablets, and mobile screens.

Technologies Used
HTML5: For semantic web structure.

CSS3: For modern layouts (Flexbox/Grid), custom animations, and styling.

JavaScript (ES6+): For dynamic task manipulation and local storage handling.

How to Use
Add a Task: Type your task into the input field at the top and press "Enter" or click the "+" button.

Complete a Task: Click the checkbox or tap the task text to cross it off your list.

Delete a Task: Click the "Delete" (trash icon) button next to any task to remove it permanently.

Filter Tasks: Use the category tabs to view "All", "Active", or "Completed" tasks.
